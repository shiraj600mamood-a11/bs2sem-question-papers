# BSc Semester Question Papers - Bhattadev University

This website is a dedicated platform for Bhattadev University BSc students, providing easy access to previous year question papers.

## Features
- **Easy Access:** A collection of BSc 1st and 2nd Semester question papers.
- **Downloadable:** Convenient PDF downloads for all subjects.
- **User-Friendly:** Simple, fast, and responsive interface.
